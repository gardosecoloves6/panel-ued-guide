# panel-ued-guide

这是一个用于归档和发布多个独立 HTML 页面的仓库。页面内容主要围绕用户界面设计、前端演示、交互原型或相关技术说明，不针对任何特定域名或具体网站。

## 项目简介

本仓库旨在整理和存放一系列自包含的 HTML 文件，便于版本管理、预览和分享。每个页面通常独立完成某项演示或说明，不依赖外部服务或特定后端环境。仓库本身不提供任何商业服务或推广内容。

## 目录说明

```
panel-ued-guide/
├── index.html          # 仓库入口页（可选）
├── pages/              # 存放各个独立 HTML 页面
│   ├── page-01.html
│   ├── page-02.html
│   └── ...
├── assets/             # 公共资源（CSS、图片、字体等）
│   ├── css/
│   ├── images/
│   └── fonts/
└── README.md           # 本文件
```

- `pages/`：主要归档目录，每个 HTML 文件为一个独立页面。
- `assets/`：存放页面间共享的样式、图片或其他静态资源。

## 页面归档说明

- 每个 HTML 页面均为独立文件，可直接在浏览器中打开运行。
- 页面命名建议使用有意义的英文或拼音，避免特殊字符。
- 若页面引用 `assets/` 下的资源，请使用相对路径。
- 页面内容不包含任何外部追踪、广告或敏感信息。

## 维护说明

- 欢迎提交 Pull Request 添加新的演示页面或修复现有页面问题。
- 提交前请确保页面在主流浏览器中基本可用。
- 不建议在页面中引入大量外部依赖，保持轻量和可离线查看为佳。
- 如添加新的公共资源，请同步更新目录结构说明。
